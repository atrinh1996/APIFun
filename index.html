About

Information on this page comes from the openFDA API, which has public FDA data on food, drugs, and devices. This page will specifically request information from the openFDA food adverse event API. This API returns data from Center for Food Safety and Applied Nutrition Adverse Event Reporting System (CAERS). I found this API from an online resource for public API's.


This page creates API requests for food (adverse food reactions) by brand name (products.name_brand) and/or data range (date_started), so users could see what adverse reactions were reports for what foods by the brand and date reported. Use the search tools above.


Applications:

  • Quality Research: This would be useful for someone wanting to research on the quality of a food brand (information that might be of interest to the public), because we can see what symptoms are reported for products from their brand and for how long/often. This can inform brands' business decision.
  • Public Health: Food/Drug/Device safety and recalls are also an important avenue of information to the general public; This can help educate consumer's choice for particular food brands.
